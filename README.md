# Antibody

A real-time PvP arithmetic puzzle game where players race to solve math challenges before antibiotics destroy their bacteria colony.

## Project Status

Currently in Phase 3 (Real-time Core) - Foundation complete with Socket.io integration and arithmetic engine.

## Tech Stack

- **Frontend**: React (Vite)
- **Backend**: Node.js (Express) + Socket.io
- **Real-time**: WebSockets for instant game synchronization

## Getting Started

### Server
```bash
cd server
npm install
npm run dev
```

### Client
```bash
cd client
npm install
npm run dev
```

Visit `http://localhost:5173` to see the game in action!
